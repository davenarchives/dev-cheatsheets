Docker - Overview

Docker is an open-source platform designed to simplify application deployment by using containers. Containers package an application and its dependencies into a single unit, ensuring consistency across development, testing, and production environments.

Run a new Container

  • docker run IMAGE
    docker run nginx

    Start a new Container from an image

  • docker run --name CONTAINER IMAGE
    docker run --name web nginx

    ...and assign it a name

  • docker run -p HOSTPORT: CONTAINERPORT IMAGE
    docker run -p 8080:80 nginx

    ...and map a port

  • docker run -p IMAGE
    docker run -p nginx

    ...and map all ports

  • docker run -d IMAGE
    docker run -d nginx

    ..and start container in background

  • docker run --hostname HOSTNAME IMAGE
    docker run --hostname srv nginx

    ...and assign it a hostname

  • docker run --add-host HOSTNAME:IP IMAGE

    ...and add a dns entry

  • docker run -v HOSTDIR:TARGETDIR IMAGE
    docker run -v ~/:/usr/share/nginx/html nginx

    ...and map a local directory into the container

  • docekr run -it --entrypoint EXECUTABLE IMAGE
    docker run -it --entrypoint bash nginx

    ...but change the entrypoint

Manage Containers

  • docker ps

    Show a list of running containers

  • docker ps -a

    Show a list of all containers

  • docker rm CONTAINER
    docker rm web

    Delete a container

  • docker rm -f CONTAINER
    docker rm -f web

    Delete a running container

  • docker container prune

    Delete stopped containers

  • docker stop CONTAINER
    docker stop web

    Stop a running container

  • docker start CONTAINER
    docker start web

    Start a stopped container

  • docker cp CONTAINER: SOURCE TARGET
    docker cp web:/index.html index.html

    Copy a file from a container to the host

  • docker cp TARGET CONTAINER: SOURCE
    docker cp index.html web:/index.html

    Copt a file from the host to a container

  • docker exec -it CONTAINER EXECUTABLE
    docker exec -it web bash

    Start a shell inside a running container

  • docker rename OLD_NAME NEW_NAME
    docker rename 096 web

    Rename a container

  • docker commit container
    docker commit web

    Create an image out of container

Manage Images

  • docker pull IMAGE[:TAG]
    docker pull nginx

    Download an image

  • docker push IMAGE
    docker push myimage:1.0

    Upload an image to a repository

  • docker rmi IMAGE

    Delete an image

  • docker images

    Show a list of all images

  • docker image prune

    Delete dangling images

  • docker image prune -a

    Delete all unused images

  • docker build DIRECTORY
    docker build .

    Build an image from a Dockerfile

  • docker tag IMAGE NEWIMAGE
    docker tag ubuntu ubuntu:18.04

    Tag an image

  • docker build -t IMAGE DIRECTORY
    docker build -t myimage .

    Build and tag an image from a Dockerfile

  • docker save IMAGE > FILE
    docker save nginx > nginx.tar

    Save an image to tar file

  • docker load -i TARFILE
    docker laod -i nginx.tar

    Load an image from a tar file

Info & Stats

  • docker logs CONTAINER
    docker logs web

    Show the logs of a container

  • docker stats

    Show stats of running containers

  • docker top CONTAINER
    docker top web

    Show processes of container

  • docker version

    Show installed docker version

  • docker inspect NAME
    docker inspect nginx

    Get detailed info about an object

  • docker diff CONTAINER
    docker diff web

    Show all modifed files in container

  • docker port CONTAIER
    docker port web

    Show mapped ports of a container
